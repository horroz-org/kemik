## Nası çalıştırıyoz oğlum bunu, bu ne?

- Sonra ``Konfigurasyon.ornek.php``'yi ``Konfigurasyon.php`` olarak kopyala, onu da değiştir, logoları mogoları herşeyi ayarla klasörlere at.
